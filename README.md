```
How to use

./shell_exporter_amd64 --help

./shell_exporter_amd64 -port 9033 -script full_path/script1.sh,full_path/script2.sh and so on OR -port=9033 -script=full_path/script1.sh,full_path/script2.sh and so on 
  -port int
    	Port number (default 9033)
  -script string
    	Comma-separated list of script paths/names (default "myscript.sh")

```
